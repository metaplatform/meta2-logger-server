# meta2-logger-server

Lightweight management server with UI for `meta2-logger` library.

Also provides REST api, check `openapi.yml` file.

## Development

```bash
# Install dependencies
npm install

# Transpile TypeScript
npm run build

# Run linter
npm run lint

# Run tests
npm test
```
